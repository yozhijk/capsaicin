## System requirements
### Windows

-  CMake 13.2 or later
-  Visual Studio 2019 or later
-  DirectX 12

### OSX

- Not supported yet

### Linux

- Not supported yet

## Build steps
```sh
git clone --recursive https://github.com/yozhijk/capsaicin.git capsaicin
mkdir build
cmake -S . -B build
cd build
make -j4
```
